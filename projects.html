Digital Etch-A-Sketch

November 2024

STM32 Microcontroller, C Programming, Embedded Systems Design

The Etch-a-Sketch Emulator project involved designing and implementing an interactive drawing system using an STM32 microcontroller and various peripheral devices. Key functionalities and features included:

1. Interactive Display: Implemented a TFT LCD display controlled via the SPI protocol to trace lines based on user input from potentiometers.

2. Analog-to-Digital Conversion: Utilized the STM32's internal ADC to translate analog voltages from two potentiometers into positional data for drawing.

3. Persistent Storage: Integrated an SD card reader using SPI to save and retrieve user drawings, enabling persistent storage of designs across sessions.

4. Motion-Based Reset and Navigation: Employed an external I2C accelerometer to detect shake motions for clearing the screen and tilting actions to navigate between saved drawings.

5. Peripheral Integration: Configured internal peripherals, including timers for display updates and DMA for efficient ADC data handling.

The Lunar Lander

November 2023

SystemVerilog, Field Programmable Gate Arrays

The Lunar Lander Project involved a sophisticated design and development of a Moon Lander utilizing SystemVerilog and a FPGA (Field Programmable Gate Arrays). The project's core comprised four integral units:

1. Memory Unit: A repository for crucial flight data.

2. Arithmetic Unit: Responsible for critical computational tasks, ensuring accurate adjustments to the lander's trajectory.

3. Control Unit: This unit was pivotal in determining the lander's status, specifically monitoring for potential crash or land scenarios.

4. Display Unit: Visually presenting key parameters such as altitude, thrust, velocity, and fuel levels on the FPGA.

For this project, I used sequential state machines for coherent workflow management and integrating modules corresponding to each unit. Additionally, I included the utilization of SystemVerilog for implementing arithmetic operations, pivotal in the accurate functioning of the Lander and integrating the modules for each unit.
